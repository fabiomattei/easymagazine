Child Theme ReadMe.txt
======================

For additional help implementing the child theme see this post:
http://buynowshop.com/2009/07/child-themes-for-everyone/

To use the child theme Shades of Darkness:

1) Copy (or move) the folder "shades-of-darkness" up one level
2) Paste the "shades-of-darkness" folder into the ../wp-content/themes folder
3) Log into your WordPress Dashboard
4) Under Appearance, click on themes
5) Locate and activate the "Shades of Darkness" theme
6) Enjoy!
